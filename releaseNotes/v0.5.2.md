# Release Notes v0.5.2

## What's New

### Improvements

- **HTTP redirect support**: The library now automatically follows HTTP redirects (301, 302, 307, 308) for both GET and POST requests
- **Enhanced test coverage**: Added comprehensive integration tests for redirect scenarios including:
  - 301 permanent redirect for GET requests
  - 302 redirect after POST (Post-Redirect-Get pattern)

### Testing Infrastructure

- **Expanded test suite**: Added 21 integration tests covering various web interaction scenarios
- **Improved test organization**: Moved inline HTML responses to separate HTML files for better maintainability
- **Separated test configurations**: Moved InteractConfig XML scripts to individual `.config` files in the Scripts directory
